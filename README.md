# PROJECT Raffle
________________

Project for create and find public raffle

**Instruction manual**

1) Loading the script
```
git clone git@github.com:gjhonic/Raffle.git
```

2) Create a .env file and copy the contents from .env.sample
 - /.env
 - /app/config/.env


3) Update Packages
```
composer update
```

4) Start migration
```
php yii migrate
```
